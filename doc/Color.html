The color module
type t
the color type, using the rgba format where: r, g and b are int between 0 and 255, and a (alpha component or opacity) is a float between 0 and 1
val rgb : int -> int -> int -> t
rgb a b c returns a color where the red component is equal to a, the green component is equal to b, the blue component is equal to c and the alpha component is equal to 1. (maximum opacity)
val rgba : int -> int -> int -> float -> t
rgb a b c d returns a color where the red component is equal to a, the green component is equal to b, the blue component is equal to c and the alpha component is equal to d (d between 0. and 1.)
